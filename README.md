# Mini ERP / Task Management System

## Tech Stack
* **OS:** Linux (Development on Windows/XAMPP)
* **Server:** Apache
* **Database:** MySQL
* **Language:** PHP 8.x (Plain/Vanilla PHP)
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **API:** REST-style JSON endpoints

## Features Implemented
* [x] Project Directory Structure Initialization
* [x] Local Environment Configuration
* [x] User Authentication & Role-based Access (Admin/User)
* [ ] Employee Management (CRUD)
* [ ] Project Management (CRUD)
* [ ] Task Management (CRUD)
* [ ] REST APIs for Core Modules
